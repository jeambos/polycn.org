PolyCN 相信集体的智慧。我们需要你的帮助来丰富这个档案馆。

## 我们欢迎什么样的内容？

### 1. 翻译与引进
优质的英文（或其他语言）文章、论文、博客或多媒体内容的中文翻译。
* *要求*：请务必确保已获得原作者授权，或原文采用开放版权协议。

### 2. 个人故事和思考 (树洞)
真实的多元关系实践经历。无论是成功的喜悦，还是踩坑的教训，亦或是困惑与反思。
* *要求*：真诚、具体。为保护隐私，请使用化名处理文中涉及的人物与地点。

### 3. 理论与科普
您自己撰写的关于关系安那其 (RA)、多边恋 (Polyamory)、沟通技巧 (如非暴力沟通) 的理论解析或实操指南。
* *要求*：我们对文章质量有一定要求，请您仔细推敲您的内容。

## 贡献原则 (Code of Conduct)

在撰写或翻译内容时，请遵循以下原则：

* **知情同意 (Consent)**：这是核心价值观。我们不支持任何形式的欺骗、隐瞒或胁迫行为。
* **包容性语言**：请使用包容性的语言，尊重不同的性别认同、性取向和关系形态。避免使用带有刻板印象或歧视性的词汇。
* **去病理化**：不要将特定的关系形态描述为“病态”或“不正常”，也不要将其神化为“更高级”的存在。

## 如何提交？

由于维护者平时有本职工作，处理消息可能不会很及时，请您谅解。以下几种方式的时效性基本相同，请选择您最方便的一种：

### 方式一：直接联系 (推荐)
您可以通过以下任一渠道直接私信我们，发送您的稿件或想法（Markdown、Word 或 PDF 格式均可）：

* **Telegram**: [@polycn](https://t.me/polycn)
* **Twitter (X)**: [@polycn_org](https://x.com/polycn_org)
* **Email**: [hi@polycn.org](mailto:hi@polycn.org)

请在消息开头注明 **[投稿]**。

### 方式二：GitHub Pull Request (极客通道)
如果您熟悉 Git 操作，欢迎直接向我们的仓库提交 PR。
*(此处建议补充您的 GitHub 仓库链接)*
