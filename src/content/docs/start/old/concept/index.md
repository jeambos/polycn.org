## 为什么需要这一页？

在关于多元关系的讨论中，  
很多争论并不是因为立场不同，而是因为**大家在用同一个词，指代完全不同的东西**。

这组页面的目标不是说服你接受任何观点，而是帮助你做到一件事：

> **至少在继续阅读之前，先听懂这些词在被用来“说什么”。**

---

## 使用说明（请先读）

- 这里的每一个词条，都尽量回答三个问题：  
  **它通常指什么 / 它不等于什么 / 常见误解来自哪里**
- 这些解释是**描述性的**，而不是规范性的  
- 不同社群、不同个人，对同一个词的使用可能并不完全一致  

如果你发现某个词让你感到不适、警惕或抗拒，  
那本身就是一个值得重视的信号。

---

## 建议的阅读顺序（不是必须）

如果你是第一次接触这些概念，可以考虑按以下顺序浏览：

1. **什么是单偶制（Monogamy）**  
2. **什么是非单偶制（Non-monogamy）**  
3. **什么是多元关系 / 多边恋（Polyamory）**  
4. **什么是开放关系（Open Relationship）**  
5. **知情同意（Informed Consent）是什么意思**  

之后再根据你的兴趣，选择其他条目。
